# 没有海外信用卡，也能低价订阅 ChatGPT / Claude

通过注册外区 Apple ID（美区、土耳其区、尼日利亚区等）+ App Store 礼品卡充值，以正规渠道低价订阅 ChatGPT Plus、Claude Pro 等 AI 服务。

> 注册美区、土耳其区、尼日利亚区等任意外区 Apple ID 的流程完全一致，只是 VPN 节点不同。本教程一次讲清楚所有区域。

**不依赖中转站、不使用黑卡灰产、不会被封号。**

## 教程目录

整个流程分为三步，按顺序操作即可：

### 1. [注册外区 Apple ID](docs/01-register-apple-id.md)
注册美区/土耳其/尼日利亚等任意外区 Apple ID，全程在 iPhone 上完成，只需要一个邮箱和中国手机号。所有区域流程一致。

### 2. [购买礼品卡并充值](docs/02-buy-gift-card.md)
通过闲鱼或 oyunfor.com 购买对应区域的 App Store 礼品卡，充值到 Apple ID 余额。

### 3. [订阅 ChatGPT / Claude 会员](docs/03-subscribe-ai.md)
用充值后的 Apple ID 在 App Store 内购订阅，正规渠道，稳定续费。

## 你需要准备什么

- 一台 iPhone 或 iPad
- 一个邮箱（Gmail、Outlook 等）
- 一个中国手机号
- 对应区域的 VPN 节点（仅注册时需要）
- 购买礼品卡的资金（闲鱼/淘宝，或有 VISA/MasterCard 可线上购买）

## 常见问题

**这个方法安全吗？会不会封号？**

App Store 内购是苹果官方支持的支付方式，ChatGPT 和 Claude 都支持通过 App Store 订阅。这是完全正规的渠道，不会因为支付方式被封号。

**比中转站贵吗？**

中转站价格虽低，但很多提供的并非官方原版模型。通过 App Store 订阅拿到的是 100% 官方服务，性价比远高于表面上看起来更便宜的中转站。

**需要一直挂 VPN 吗？**

不需要。只有注册 Apple ID 时需要对应区域的节点，注册完成后日常使用不需要。
